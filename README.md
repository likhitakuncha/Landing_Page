# Landing Page Project

## Overview

This project is a simple and responsive Landing Page created using HTML and CSS. It is designed to provide a clean and attractive user interface while demonstrating fundamental web development skills.

## Features

* Responsive Navigation Bar
* Hero Section with Call-to-Action Button
* Features Section (Learn, Build, Grow)
* About Section
* Footer Section
* Clean and Modern Design
* CSS Styling and Hover Effects

## Technologies Used

* HTML5
* CSS3

## Project Structure

```
Landing-Page/
│
├── index.html
├── style.css
└── README.md
```

## How to Run

1. Download or clone the project.
2. Open the project folder.
3. Open `index.html` in any web browser.

## Learning Outcomes

Through this project, I learned:

* Structuring web pages using HTML.
* Styling web pages with CSS.
* Creating responsive layouts.
* Using Flexbox for alignment.
* Designing user-friendly interfaces.
